# Useful Shell Scripts

### Usage (Linux)

  1. Give permission to run the script in your terminal

  ```
  chmod +x filename.sh
  ```

  2. Run Shell Script

  ```
  ./filename.sh
  ```

  3. Finish!


### Usage (MacOS)

  1. Give permission to run the script in your terminal

  ```
  chmod 755 filename.sh
  ```

  2. Run Shell Script

  ```
  ./filename.sh
  ```

  3. Finish!
